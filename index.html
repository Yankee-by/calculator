Калькулятор


Простой и компактный калькулятор, который спокойно разместится на вашем сайте.

Имеет внешний вид, схожий со встроенным в Windows 8.1.

Для развертывания достаточно передать методу calculateAll.show(DOM, path) DOM элемент в котором он будет размещаться (станет родителем). Path - путь к папке с файлами картинок фона (пример "../calculateImg"). По умолчанию эти файлы хранятся в 'img'
Если методу не передавать DOM элемент, то калькулятор отрисуется в body.

Тело имеет точный размер (height: '312px', width: '215px').
Если вы хотите убрать его, то повторно вызовите calculateAll.show(DOM).

Тестировалось в Chrome 47, IE 11, Firefox 43 и Opera 12.17.
Для работы необходима предзагрузка библиотеки jQuery.js.
